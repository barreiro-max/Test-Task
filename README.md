# IOS Test Task

## 🚀 Quick Start

The project is completely self-contained and requires no third-party dependencies or external package managers.

## How to start
1. Copy the repository URL (e.g., `https://github.com/barreiro-max/Test-Task.git`).
2. Open **Xcode**.
3. In the welcome window, click **"Clone an existing project"** (or go to the top menu and select **Source Control > Clone...**).
4. Paste the repository URL into the text field and click **Clone**.
5. Choose a local folder on your Mac to save the project. Xcode will download and open it automatically.

## 🛠 How to Test & Run
You can view and test using two standard methods:
Xcode Live Preview: Open the target view file (e.g., ImageSliderView.swift) and activate the Canvas (Cmd + Option + Return). The component is fully interactive right inside the preview canvas.
Simulator / Physical Device: Select your target scheme and press Cmd + R to build and run the application.

## Compatibility & Requirements
- Xcode Version 26
Target Devices (Tested & Verified):
- iPhone 16 SE (simulator)
- iPhone 17 Pro Max (simulator)
- IPhone 13 Pro Max (device)
